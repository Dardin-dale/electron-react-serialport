## Features

* Electron 4
* Electron Builder
* React 16
* Webpack 4
* Babel 7
* Hot Module Replacement

## Installation

* `git clone https://github.com/hamzaak/electron-react-webpack-boilerplate.git`
* cd electron-react-webpack-boilerplate
* npm install
* npm start

## Packing for distribution

To package the app fro windows platform:

`npm run dist`
